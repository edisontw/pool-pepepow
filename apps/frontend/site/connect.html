3-step setup

1 Prepare a PEPEPOW wallet address

Use your own PEPEPOW wallet address as the miner username. A worker name can be appended after a dot.

2 Download HTN Miner and include --pepepow

Use the current HTN Miner instructions from the full PEPEPOW mining guide. Avoid outdated miner examples from older algorithms.

3 Connect and check your shares

Start mining on this pool, then open Miner Lookup to check accepted shares for your wallet.

CPU example

./hoo_cpu -o stratum+tcp://pool.pepepow.net:39333 -u YOUR_WALLET_ADDRESS.worker1 -t 1 -p x --pepepow

Replace YOUR_WALLET_ADDRESS.worker1 with your own PEPEPOW wallet address and optional worker name.

GPU example

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:~/hoo_gpu/libs
./hoo_gpu -o stratum+tcp://pool.pepepow.net:39333 -u YOUR_WALLET_ADDRESS.worker1 -gpu-id 0 -p x --pepepow

GPU setup details can change by driver, library path, and miner release. Use the full guide for current download and setup details.

Pool notes

  • Use YOUR_WALLET_ADDRESS or YOUR_WALLET_ADDRESS.worker as the username.
  • Password can usually be x.
  • Accepted shares may take up to a minute to appear on Miner Lookup.
  • Pool-side accepted shares are activity records, not a reward guarantee.
  • The public website does not expose daemon RPC, wallet RPC, submit controls, or operator admin tools.
